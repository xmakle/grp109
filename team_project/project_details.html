About

Project Description:

Create a personal website that serves as a journal or portfolio to document our web development journey.

Project Purpose:

Show learning progress using technologies studied in the course along with timeline showing the learning progress visually.

What the Group Project Assignment Is

Our group project involves collaboratively building an interactive, personal, and reflective site. The aim is to demonstrate core web development skills — including HTML, CSS, JavaScript, and good design principles — while telling our story as learners.

What Inspired the Site

We were inspired by the idea of creating a digital space that both tracks our progress and helps us reflect on what we've learned felt meaningful and motivating.

What the Site's Purpose Is

The site's purpose is to act as a personal learning archive. It documents coding growth, showcases mini-projects, and builds a foundation for future portfolios or job-seeking tools.

Intended Audience

The intended audience includes:

  • Our instructors and peers
  • Fellow developers on similar learning paths

What It Does

  • Visually maps our web development learning progress
  • Displays journal entries and reflections
  • Highlights key technologies we've used

How the Site Fulfills the Audience’s Needs

  • A clear record of learning and improvement
  • Insight into our growth and motivation as developers

How and Why the Site Is Relevant Today

This project aligns with the modern demand for self-starting, tech-savvy professionals and highlights how web tools can be used to tell compelling personal stories.
